# Project: infoLib Library Management System

## Vision
A modern, high-parity desktop application (Tauri-based) that replicates the core functionality and user experience of the legacy infoLib system while leveraging modern web technologies.

## Tech Stack
- **Frontend**: React (Vite) + TypeScript
- **Backend**: Tauri (Rust) + Prisma (ORM)
- **Database**: PostgreSQL (Supabase/Local)
- **Styling**: Tailwind CSS + Custom Bevel Utilities

## Core Modules
1. **Cataloging**: Comprehensive record management and authority control.
2. **Circulation**: Lending, returns, and fine management.
3. **Inventory/Holdings**: Physical copy tracking and location mapping.
4. **User Management**: Authentication and role-based access control.
